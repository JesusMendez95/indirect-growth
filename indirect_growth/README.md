# Indirect Growth

A personal development mobile app with Flutter and Firebase.

## Features
- Habit tracking with streak counter
- RPG-style progression system
- Multimedia journal entries
- Identity tracking (beliefs, values, virtues)
- Time capsules and reflections

## Getting Started

### Prerequisites
- Flutter SDK
- Firebase project
- Dart

### Setup

1. Install dependencies:
```bash
flutter pub get
```

2. Configure Firebase:
```bash
dart pub global activate flutterfire_cli
flutterfire configure
```

3. Run the app:
```bash
flutter run
```

### Testing
```bash
flutter test
```
